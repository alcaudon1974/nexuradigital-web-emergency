Política de privacidad de Nexura Scan España

Aplicación: Nexura Scan España · Última actualización: 26 de junio de 2026

Nexura Scan España está diseñada con enfoque local-first: permite crear expedientes, escanear o importar imágenes, ejecutar OCR y generar documentos PDF/TXT en el propio dispositivo. No requiere cuenta de usuario ni sube documentos a una nube propia.

Responsable y contacto

Responsable del servicio: Nexura Digital.

Correo de contacto para privacidad y soporte: nexuradigital2026@gmail.com.

Web: https://nexuradigital.es/.

Datos que trata la app

La app puede tratar localmente, dentro del dispositivo, los siguientes datos cuando el usuario decide usarlos:

  • Imágenes o documentos que el usuario importa mediante el selector de Android.
  • Fotografías capturadas con la cámara para crear páginas escaneadas.
  • Texto OCR extraído de esas imágenes para mostrarlo y exportarlo.
  • Metadatos locales del expediente: título, categoría, fecha y número de páginas.
  • Registros técnicos locales de errores básicos, si se producen, para diagnóstico interno de la app.

Datos que no se recogen por defecto

  • No hay cuenta de usuario.
  • No se suben documentos a servidores propios de Nexura Digital.
  • No se venden datos personales.
  • No se usa publicidad dentro de la app.
  • No se integra analítica de terceros en esta versión.
  • No se solicita ubicación.

Permisos

  • Cámara: se usa solo cuando el usuario pulsa “Escanear con cámara”.
  • Documentos/imágenes: se accede únicamente a los archivos elegidos por el usuario mediante el selector del sistema Android.

OCR y procesamiento local

La app utiliza reconocimiento de texto integrado en Android/ML Kit para extraer texto de imágenes. El resultado OCR se guarda localmente en el almacenamiento interno de la aplicación, junto con el expediente correspondiente.

Almacenamiento y eliminación

Los expedientes, imágenes internas y textos OCR se guardan en el almacenamiento interno de la app. El usuario puede borrar páginas o expedientes desde la interfaz. Al desinstalar la app, Android elimina el almacenamiento interno asociado a la aplicación.

Exportación y compartición

Cuando el usuario exporta un PDF o TXT, Android muestra el selector de aplicaciones para compartir. La app externa elegida por el usuario puede tratar esos archivos conforme a su propia política de privacidad. Nexura Scan España no controla el tratamiento realizado por apps externas una vez que el usuario decide compartir.

Menores

La app está orientada a organización documental general. No está diseñada para recopilar datos de menores ni para crear perfiles de usuario.

Cambios en esta política

Si en futuras versiones se añaden cuentas, nube, sincronización, analítica, publicidad u otros tratamientos de datos, esta política se actualizará antes de publicar esos cambios.
